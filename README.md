                                                      📱 Smartphone Sales Analysis Dashboard
📌 Project Overview

This project focuses on analyzing smartphone sales data using data analysis and visualization techniques in Python. The objective of the project is to explore relationships between smartphone price, sales, and ratings using different graphical visualization methods.

The project demonstrates how NumPy, Pandas, and Matplotlib can be used together for data manipulation, statistical analysis, and data visualization.

The dashboard presents multiple visual analytics to better understand smartphone market trends and performance.

🎯 Objectives

Analyze smartphone sales data using Python

Perform data filtering, sorting, and grouping using Pandas

Apply NumPy statistical functions for numerical analysis

Visualize smartphone sales patterns using multiple graph types

Demonstrate advanced data visualization dashboards using Matplotlib

🗂 Dataset

The dataset contains information about different smartphones including their brand, price, sales volume, and customer ratings.

Column	Description

Phone	Smartphone model identifier

Brand	Smartphone brand

Sales	Total units sold

Price	Price of the smartphone

Rating	Customer rating of the smartphone

Example Data:

Phone	Brand	Sales	Price	Rating
A	Samsung	500	20000	4.2

B	Apple	650	70000	4.6

C	Samsung	700	25000	4.3

📊 Visualizations

The dashboard contains 6 different visualizations:

1️⃣ Price vs Sales (Scatter Plot)

Shows the relationship between smartphone price and sales volume.

2️⃣ Phone Sales (Horizontal Bar Chart)

Displays sales performance of different smartphone models.

3️⃣ Sales Stem Plot

Shows individual sales values using a stem representation.

4️⃣ Sales & Rating Stack Plot

Visualizes combined trends of smartphone sales and ratings.

5️⃣ Sales Distribution (Violin Plot)

Displays the distribution and spread of smartphone sales.

6️⃣ Sales Density (Hexbin Plot)

Represents sales concentration using hexagonal binning.
